Welcome to Moengage data import

The data imported will be added to Moengage users segments. Upon viewing an user, the fields: name, first_name, last_name, email, age, gender, mobile in the data will be displayed Tracked Standard Attribute section as well as the main user info area (First Name, Last Name, Email and Mobile). All other fields in the dataset will be under Tracked Custom Attribute.

Data will be filtered according to the date of each record. This means that only records with the date later than the last updated date will be selected for import.
